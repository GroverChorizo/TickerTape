## **2. Liquidation cascade** - https://papers.ssrn.com/sol3/Delivery.cfm/5611392.pdf?abstractid=5611392&mirid=1 - a chain reaction in leveraged financial marketsвҖ”especially common in cryptocurrencyвҖ”where the forced closure of leveraged positions triggers further price declines, which in turn cause more positions to be liquidated.  This creates a self-reinforcing downward spiral of selling pressure and increasing volatility.
    Example: Liquidation Cascade Alert System in Python with Textual
from textual.app import App, ComposeResult
from textual.widgets import Label, Header, Footer
from textual.containers import Vertical
from textual.reactive import reactive
import asyncio
import random  # Simulate liquidation data

class CascadeAlertApp(App):
    CSS = """
    #alert { color: red; background: white; padding: 2; }
    #normal { color: green; }
    """
    
    # Reactive attribute to track liquidation state
    liquidation_volume = reactive(0.0)
    alert_active = reactive(False)

    def compose(self) -> ComposeResult:
        yield Header()
        yield Vertical(Label("Liquidation Cascade Monitor"), id="main")
        yield Footer()

        # Threshold for cascade alert (e.g., > $300M in 5s)
        if volume > 300_000_000:
            if not self.alert_active:
                self.notify("рҹҡЁ LIQUIDATION CASCADE DETECTED!", severity="error")

            self.query_one("#main").remove_class("error")

## **3. Funding rate extreme** - 
## What Are Funding Rate Extremes?

### On Hyperliquid:
- Funding is paid **hourly**. 
- Formula:  
  `Funding Rate = Average Premium Index + clamp(0.000125 - Premium Index, -0.0005, 0.0005)`  
  (Fixed interest rate = 0.01% per 8h вҶ’ ~0.00125% per hour)
- Capped at **Вұ4% per hour** (extremely rare). 
- **Positive rate**: Longs pay shorts вҶ’ bullish bias. 
- **Negative rate**: Shorts pay longs вҶ’ bearish bias.

Use the official **Hyperliquid Data Layer API** to fetch real-time funding rates and trigger alerts in your Textualize dashboard.

def detect_anomaly(data, history):
    alerts = []

    # Price spike (5m vs 1h MA)
    short_ma = data["price"]
    long_ma = sum(h["price"] for h in history[-12:]) / 12  # 1h
    if abs(short_ma - long_ma) / long_ma > 0.05:
        alerts.append("рҹҡЁ PRICE SPIKE")

    # Volume surge
    avg_volume = sum(h["volume"] for h in history[-12:]) / 12
    if data["volume"] > 3 * avg_volume:
        alerts.append("рҹҡЁ VOLUME SURGE")

    # Funding extreme
    if abs(data["funding_rate"]) > 0.0001:
        alerts.append("рҹҡЁ EXTREME FUNDING")

    # Liquidity drop
    prev_bid, prev_ask = history[-1]["bid_depth"], history[-1]["ask_depth"]
    if prev_bid and (abs(data["bid_depth"] - prev_bid) / prev_bid > 0.5):
        alerts.append("рҹҡЁ BID LIQUIDITY DRAIN")
    if prev_ask and (abs(data["ask_depth"] - prev_ask) / prev_ask > 0.5):
        alerts.append("рҹҡЁ ASK LIQUIDITY DRAIN")

    return alerts
```

# Or custom implementation:
def create_sparkline(data: List[float], width: int = 20) -> str:
    """Generate Unicode Braille sparkline"""
    # Normalize to 0-7 range for Braille characters
    normalized = [(x - min(data)) / (max(data) - min(data)) * 7 for x in data]
    braille_base = 0x2800  # Base Braille Unicode
    
    sparkline = ""
    for val in normalized[-width:]:
        sparkline += chr(braille_base + int(val))
    
    return sparkline
```

### **Multi-Window/Detached Panel Support**

**Concept:** Power users with multiple monitors can detach panels into separate terminal windows.

**Implementation:**
```bash
# Terminal 1 (Main Dashboard)
$ hlterm --main

$ hlterm --panel orderbook --follow btc

$ hlterm --panel whales --fullscreen

**Behavior:**
- Panels communicate via IPC (shared memory or Redis)
- Data synced in real-time across windows
- Each window maintains own layout state
- Close detached window вҶ’ panel returns to main

**Use Cases:**
- **Multi-Monitor Setup:** Main dashboard + dedicated orderbook screen
- **Focus Mode:** Single panel fullscreen on secondary display
- **Recording/Streaming:** Detach specific panel for screen capture

def get_funding_color(yearly_rate: float) -> tuple:
    """Determine color based on funding rate severity"""
    if yearly_rate > 50:
        return 'black', 'on_red'        # Extreme positive
    elif yearly_rate > 30:
        return 'black', 'on_yellow'     # High positive
    elif yearly_rate > 5:
        return 'black', 'on_cyan'       # Elevated positive
    elif yearly_rate > -10:
        return 'black', 'on_green'      # Normal range
    else:
        return 'black', 'on_light_green'  # Negative (shorts paying)

### **Arbitrage Detection Logic**
```python
def detect_funding_arbitrage(rates: dict) -> List[dict]:
    """
    Identify funding rate arbitrage opportunities.
    
    Returns list of opportunities with:
    - Symbol
    - Long exchange (lowest rate)
    - Short exchange (highest rate)
    - Spread (difference)
    - Expected annual profit
    """
    opportunities = []
    
    for symbol in rates.keys():
        exchanges = rates[symbol]
        
        # Find min/max rates
        min_exchange = min(exchanges, key=lambda x: x['rate'])
        max_exchange = max(exchanges, key=lambda x: x['rate'])
        
        spread = max_exchange['rate'] - min_exchange['rate']
        
        # Only flag if spread >0.05% (0.0005)
        if spread > 0.0005:
            annual_profit = (spread * 3 * 365) * 100  # 3 funding periods per day
            
            opportunities.append({
                'symbol': symbol,
                'long': min_exchange['name'],
                'short': max_exchange['name'],
                'spread': spread,
                'annual_profit_pct': annual_profit,
                'confidence': 'HIGH' if spread > 0.001 else 'MEDIUM'
            })
    
    return sorted(opportunities, key=lambda x: x['spread'], reverse=True)
```

def get_funding_color(yearly_rate: float) -> tuple:
    """Determine color based on funding rate severity"""
    if yearly_rate > 50:
        return 'black', 'on_red'        # Extreme positive
    elif yearly_rate > 30:
        return 'black', 'on_yellow'     # High positive
    elif yearly_rate > 5:
        return 'black', 'on_cyan'       # Elevated positive
    elif yearly_rate > -10:
        return 'black', 'on_green'      # Normal range
    else:
        return 'black', 'on_light_green'  # Negative (shorts paying)
```

### **Arbitrage Detection Logic**
```python
def detect_funding_arbitrage(rates: dict) -> List[dict]:
    """
    Identify funding rate arbitrage opportunities.
    
    Returns list of opportunities with:
    - Symbol
    - Long exchange (lowest rate)
    - Short exchange (highest rate)
    - Spread (difference)
    - Expected annual profit
    """
    opportunities = []
    
    for symbol in rates.keys():
        exchanges = rates[symbol]
        
        # Find min/max rates
        min_exchange = min(exchanges, key=lambda x: x['rate'])
        max_exchange = max(exchanges, key=lambda x: x['rate'])
        
        spread = max_exchange['rate'] - min_exchange['rate']
        
        # Only flag if spread >0.05% (0.0005)
        if spread > 0.0005:
            annual_profit = (spread * 3 * 365) * 100  # 3 funding periods per day
            
            opportunities.append({
                'symbol': symbol,
                'long': min_exchange['name'],
                'short': max_exchange['name'],
                'spread': spread,
                'annual_profit_pct': annual_profit,
                'confidence': 'HIGH' if spread > 0.001 else 'MEDIUM'
            })
    
    return sorted(opportunities, key=lambda x: x['spread'], reverse=True)
```
